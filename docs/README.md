# 戸山流居合道会ホームページ - ドキュメント

このディレクトリには、プロジェクトの要件定義、設計書、議事録などのドキュメントを格納します。

## ディレクトリ構成

```
docs/
├── requirements/     # 要件定義書
├── design/          # デザイン・UI/UX関連
├── architecture/    # システム設計・アーキテクチャ
├── meeting-notes/   # 議事録・打ち合わせメモ
└── assets/          # ドキュメント用画像・図表
```

## ドキュメント管理ルール

- Markdownフォーマットを使用
- ファイル名は日付プレフィックス付き推奨（例: `2025-10-03_initial-requirements.md`）
- 重要な決定事項は必ず文書化
- バージョン管理はGitで実施
